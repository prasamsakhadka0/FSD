What is Bootstrap?

Bootstrap is a free and open-source CSS framework used to build responsive and mobile-first websites. It was originally developed by Twitter and is now one of the most popular frontend frameworks in the world. Bootstrap provides ready-made components such as buttons, cards, progress bars, badges and grids that developers can use to build professional websites quickly.

Instead of writing all CSS from scratch, Bootstrap allows developers to use pre-built classes directly in their HTML. This saves time and ensures consistent styling across all browsers and screen sizes.

Key Bootstrap Features

  • Grid System – Splits the page into 12 columns for responsive layouts.
  • Components – Ready-made cards, buttons, alerts, badges and progress bars.
  • Mobile-First – Designed to work on small screens first and scale up.
  • Utility Classes – Short classes like mt-3 and text-center for quick styling.
  • No installation needed – Just add a CDN link to the HTML head.

How to Include Bootstrap

Bootstrap can be added to any HTML page by including this CDN link inside the head section of the document. No downloading or installation is needed: 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
      rel="stylesheet">

The Bootstrap Grid System

The Bootstrap grid divides the page into 12 columns. Columns can be combined to create different layouts. For example col-md-6 creates two equal columns and col-md-4 creates three equal columns:

Class What it does
container Centres the content and adds padding on the sides
row Creates a horizontal row for columns to sit inside
col-md-4 Takes up 4 out of 12 columns on medium screens
col-md-6 Takes up 6 out of 12 columns, creates two equal halves
col-12 Takes up the full width of the row on all screen sizes

Bootstrap Code Snippet

<div class="card shadow">
    <div class="card-body text-center p-4">
        <h4 class="card-title">Priyanshu Khadka</h4>
        <p class="card-text text-muted">Full Stack Student</p>

        <span class="badge bg-primary">HTML</span>
        <span class="badge bg-success">CSS</span>

        <div class="progress mb-2">
            <div class="progress-bar bg-danger" style="width:90%">
                90%
            </div>
        </div>

        <button class="btn btn-primary">View Portfolio</button>
    </div>
</div>

<div class="alert alert-success">
    Currently enrolled in Full Stack Development
</div>

Explanation of the Code

The card and card-body classes create the white box with padding. The shadow class adds a drop shadow to make the card stand out on the page. The text-center class centres all the content inside and p-4 adds extra padding around everything. The badge class with colours like bg-primary and bg-danger creates the small skill labels. For the progress bars the progress class is used on the outer div and progress-bar on the inner div. The width is set using an inline style to control how full each bar appears. The btn class with btn-primary styles the blue button and btn-outline-secondary creates the outlined button. The alert class with alert-success makes the green box and alert-info makes the blue one.

Bootstrap vs Custom CSS

Both Bootstrap and custom CSS can be used to style a webpage but they each have different advantages and disadvantages:

Feature Bootstrap Custom CSS
Speed Quick, classes are ready to use Takes longer to write everything
Responsive layout Built in automatically Have to write media queries manually
Browser support Works the same in all browsers Sometimes looks different in each browser
Customisation Can be limited by the framework Full control over every style
File size Larger due to the full framework Smaller, only what is needed

Conclusion

Bootstrap is a powerful and widely used CSS framework that makes building responsive and professional websites much faster. Through this demonstration, an understanding of the Bootstrap grid system, card components, badges, progress bars, buttons and alerts has been shown. Using Bootstrap alongside a custom CSS file allows developers to build consistent and attractive web pages without writing everything from scratch.
