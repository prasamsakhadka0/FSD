What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to control the appearance and layout of web pages. While HTML provides the structure of a webpage, CSS is responsible for colours, fonts, spacing, positioning and overall design.

CSS allows developers to create visually attractive websites and improve the user experience. Modern websites rely heavily on CSS for responsive layouts, animations and professional styling.

Types of CSS

  • Inline CSS – Applied directly inside an HTML element.
  • Internal CSS – Written inside a style tag within the HTML file.
  • External CSS – Stored in a separate CSS file and linked to webpages.

This portfolio uses External CSS through the style.css file because it makes styling easier to manage across multiple pages.

CSS Selectors

CSS selectors are used to target specific HTML elements. The most common selectors are element selectors, class selectors and ID selectors. 

p {
    color: blue;
}

.card {
    background: white;
}

#header {
    text-align: center;
}

CSS Box Model

The CSS Box Model is one of the most important concepts in CSS. Every element consists of content, padding, border and margin.

  • Content – The text or image inside the element.
  • Padding – Space between content and border.
  • Border – Surrounds the content and padding.
  • Margin – Space outside the border.

CSS Code Snippet

.profile-card{
    background:white;
    padding:25px;
    border-radius:15px;
    text-align:center;
    box-shadow:0 4px 15px rgba(0,0,0,0.15);
}

Explanation of the Code

The profile card uses several CSS properties. The background property changes the card colour, padding creates internal spacing, border-radius creates rounded corners and box-shadow adds depth to the design. These properties demonstrate how CSS can transform plain HTML into an attractive user interface.

Responsive Design

Responsive design ensures that websites work correctly on desktops, tablets and smartphones. CSS provides tools such as media queries, Flexbox and Grid layouts to help developers create responsive websites.
