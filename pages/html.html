What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language which is used to create web pages and applications. HTML provides the structure of a webpage by using different elements such as headings, paragraphs, images, links, tables, lists and forms.

Every website on the internet uses HTML as its foundation. While HTML creates the structure of a webpage, CSS is used for styling and JavaScript is used to add interactivity and dynamic behaviour.

Key HTML Elements

  • Headings (<h1> to <h6>)
  • Paragraphs (<p>)
  • Links (<a>)
  • Images (<img>)
  • Lists (<ul> and <ol>)
  • Tables (<table>)
  • Forms (<form>)

Explanation of the Code

The form uses several important HTML elements. The label element identifies the purpose of each field. The input element is used to collect text, email, radio button, checkbox, tel and file upload data. The select element creates a dropdown menu for both country code and course selection. The textarea element allows multi-line text input for the address field, while the button element submits the form.
